============
Installation
============

To begin using the ``py-pkg-logging`` API, install from PYPI (recommended) or GitHub.

Install via pip (recommended)
"""""""""""""""""""""""""""""
.. code-block:: python

    pip install py-pkg-logging


GitHub (Developer version)
""""""""""""""""""""""""""
.. code-block:: python

    # Install the developer version via GitHub
    
    git clone https://github.com/mvinyard/py-pkg-logging.git; cd ./py-pkg-logging;
    pip install -e .
